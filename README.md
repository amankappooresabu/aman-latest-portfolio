# Portfolio

A minimal, fast, and responsive personal portfolio built with Next.js and Bun.

## Tech Stack

- **Framework** — Next.js 15 (App Router)
- **Runtime** — Bun
- **Styling** — Tailwind CSS
- **Components** — shadcn/ui
- **Theme** — next-themes (dark/light mode)
- **Icons** — Lucide React, Simple Icons


```

## Getting Started
```bash
# Install dependencies
bun install

# Run development server
bun dev

# Build for production
bun run build

# Start production server
bun start
```

## Sections

- **Intro** — Name, tagline and profile photo
- **About** — Short bio
- **Work Experience** — Companies, roles and duration
- **Education** — College, degree and duration
- **Skills** — Tech stack as tags
- **Projects** — Project cards with preview, description, tags and links
- **Contact** — Email and social links

## Features

- Dark and light mode with system preference detection
- Smooth scroll navigation via fixed bottom dock
- Fully responsive
- Built with Bun for fast installs and dev server
